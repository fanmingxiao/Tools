# CATIA V5 测量点导出工具

## 功能概述

这是一个用于 CATIA V5 的 VBS 自动化脚本工具，可以将选中的测量点数据批量导出为 Excel 报告，包含坐标信息和可视化截图。

## 功能特点

- 📐 **自动提取测量点坐标** (X, Y, Z)
- 📸 **自动截图** 每个测量点的视图
- 📊 **生成专业Excel报告**，包含：
  - 序号
  - 测量点截图
  - X/Y/Z 坐标值
  - 测量点名称
  - 所属元素信息
- 🎯 **支持批量处理** 多个选中的测量点

## 使用条件

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 7/10/11 |
| CATIA 版本 | CATIA V5 R19 或更高版本 |
| Office | Microsoft Excel 2007 或更高版本 |
| 权限 | 需要 CATIA 和 Excel 的运行权限 |

## 使用方法

### 1. 在 CATIA 中选择测量点

1. 打开 CATIA V5 并加载包含测量点的文档
2. 在结构树或图形区域中**选中需要导出的测量点**（可多选）
3. 支持的点类型：
   - Point（点）
   - HybridShapePoint（混合造型点）
   - Measure（测量点）
   - 其他可测量点对象

### 2. 运行脚本

**方式一：直接双击运行**
```
双击 "测量点导出工具.vbs"
```

**方式二：通过命令行运行**
```batch
cscript "测量点导出工具.vbs"
```

**方式三：添加到 CATIA 工具栏（推荐）**
1. 在 CATIA 中点击 **工具(Tools)** → **自定义(Customize)**
2. 选择 **命令(Commands)** 选项卡
3. 点击 **新建(New)** 创建一个宏命令
4. 选择 VBS 脚本路径，添加到工具栏

### 3. 导出流程

1. 运行脚本后，会显示操作提示对话框
2. 点击 **确定** 继续
3. 在弹出的文件夹浏览器中选择**导出文件夹**
4. 脚本会自动：
   - 提取每个选中点的坐标信息
   - 截取当前视图保存为图片
   - 生成包含数据的 Excel 报告
5. 导出完成后会自动打开 Excel 文件

## 输出文件说明

导出完成后，在选择的文件夹中会生成：

```
导出文件夹/
├── 测量点数据报告.xlsx    # Excel 报告文件
└── Screenshots/           # 截图文件夹
    ├── Point_1.png
    ├── Point_2.png
    └── ...
```

## 注意事项

1. **选择对象**：运行脚本前必须先在 CATIA 中选中测量点，否则脚本会提示无选中对象

2. **截图质量**：截图使用 CATIA 当前视图，建议调整视图角度和缩放比例以获得最佳截图效果

3. **坐标单位**：坐标值使用 CATIA 文档的当前单位（通常为毫米 mm）

4. **Excel 版本**：脚本使用 `.xlsx` 格式保存，需要安装支持该格式的 Excel 版本

5. **权限问题**：
   - 确保有写入导出文件夹的权限
   - 确保 CATIA 和 Excel 的 COM 接口可访问
   - 如遇权限问题，尝试以管理员身份运行

## 常见问题

### Q: 脚本提示"无法连接到CATIA"
A: 请确保 CATIA V5 已启动，且文档已打开。某些安全软件可能会阻止 COM 连接。

### Q: 截图显示空白或不完整
A: 脚本使用当前 CATIA 视图进行截图。请在运行脚本前调整视图到合适的角度和缩放比例。

### Q: 坐标显示为 0
A: 可能是点对象类型不支持直接读取坐标。确保选中的是真正的 Point 对象，而非其他几何元素的参考点。

### Q: Excel 文件无法打开
A: 请检查是否已安装 Microsoft Excel，以及是否有足够的磁盘空间保存文件。

## 技术说明

### 使用的 CATIA API

- `CATIA.Application` - CATIA 应用程序对象
- `CATIA.ActiveDocument` - 当前活动文档
- `Selection` - 选择集对象
- `Measurable` - 测量接口
- `Viewer.CaptureToFile` - 截图功能

### 使用的 Windows 组件

- `Scripting.FileSystemObject` - 文件系统操作
- `Shell.Application` - 文件夹浏览对话框
- `Excel.Application` - Excel 自动化

## 自定义修改

如需修改脚本行为，可以编辑 VBS 文件中的以下参数：

```vbscript
' 修改截图缩放比例（默认 0.8）
dScale = 0.8

' 修改行高（默认 80 像素）
objWorksheet.Rows(iRow).RowHeight = 80

' 修改坐标精度（默认 4 位小数）
objWorksheet.Cells(iRow, 3).Value = Round(x, 4)
```
