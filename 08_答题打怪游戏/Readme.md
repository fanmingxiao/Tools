# 答题打怪游戏

## 功能简介
这是一个基于网页（HTML + CSS + JS）的轻量级第一视角答题打怪小游戏。
玩家通过回答题库中的选择题来击败不同的怪物，每答对一题会对怪物造成伤害。答题正确率达标后即可战胜怪物并进入下一关挑战更强的对手。

## 使用条件
- **无需安装任何环境**，纯前端实现。
- 只需要使用任何现代浏览器（Chrome、Edge、Safari 等）打开即可。

## 使用方法
1. 双击运行 `index.html`，在浏览器中打开游戏。
2. 在主界面点击「导入题库 (CSV)」。
3. 选取准备好的 `.csv` 题库文件（同目录下提供了一个 `demo_questions.csv` 作为示例）。
4. 导入成功后，点击「开始冒险」。
5. 在限定时间内（30秒/题）阅读题目并选择正确的选项。
6. 答对扣减怪物血量，答错不扣血，完成后会根据正确率判定是否通过当前关卡。

## 题库格式说明
如需自定义题库，请确保 CSV 文件包含以下列头：
- `question`: 题目内容
- `option_a`: 选项A
- `option_b`: 选项B
- `option_c`: 选项C
- `option_d`: 选项D
- `answer`: 正确答案（填A/B/C/D）
