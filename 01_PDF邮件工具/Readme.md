# PDF邮件工具

## 功能简介
这是一个基于 Windows 批处理（.bat）和 VBScript 的自动化小工具。
其主要功能是：
1. 自动收集当前目录下的所有 `.pdf` 文件。
2. 调用本地的 **Microsoft Outlook** 创建一封新邮件，自动填入指定的正文，并将这些 PDF 文件作为附件添加。
3. 将处理过的 PDF 文件自动移动到以今天日期（YYYYMMDD）命名的新文件夹中进行归档。

## 使用条件
- 操作系统：Windows
- 必须安装并配置好 **Microsoft Outlook** 客户端。

## 使用方法
1. 将需要发送的 PDF 文件与 `发送PDF邮件.bat` 放在同一个目录下。
2. 双击运行 `发送PDF邮件.bat`。
3. 脚本会自动启动 Outlook 并生成邮件（包含相关附件和说明）。
4. 核对邮件内容及收件人后，在 Outlook 中手动点击发送。
5. 脚本运行完成后，原目录下的 PDF 文件会被归档到当前日期对应的文件夹中。
