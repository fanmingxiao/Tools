# 13_Kindle批量解码工具

## 痛点引入
在 Kindle 国内服务退场、设备老化报废的情况下，如何拯救我们高价购买囤积的大量 `.azw3` 或者包含 Amazon DRM 所属机器绑定的数字资产？这是一个提供**无损解密并输出跨平台开放格式**的高效解决方案。

## 工作原理
利用 Python `pycryptodome` 加解密核心计算，并辅以经典防滥用解包算法模块集群（DeDRM核心）；构建一套能根据唯一设备特征对本地书库做高维度多向脱壳的集成。此外，为了降低门槛与适配自动化量产办公场景，程序提供了完备的多线程架构以及 `Tkinter` 生成的光鲜可视化带交互状态窗格（GUI），可一键读取大量电子书并行脱去 DRM 。

## 使用条件与核心依赖
- **Python** 环境（由于项目强制使用了现代化 Python 包管工具进行，请确认具有 `uv` 解释器管线环境）：调用 `uv run python app.py` 会自动解析或补全缺失的依赖库（包括 `pycryptodome` ）。
- **设备密级验证**：您在进行批处理时，需获知并填入属于您名下购买账号或当初推送到设备上的唯一 16 位序列识别码 `Kindle Serial Number`（通常为 `B0...` 开头的一长串）。 
- **工具入口**：`app.py`  (核心控制执行台)

## 使用方法
1. 从终端进入本目录，输入指令：
   ```bash
   uv run python app.py
   ```
2. 在弹出的安全解码站视窗中指定【扫描目标目录】与填写【Kindle 序列号 (Serial)】。
3. 点击发射。程序底层 `core_decryptor.py` 控制轴开始呼叫内嵌资源并行计算和解码还原文件，将安全无毒的 `.azw3` 原始书本吐露至特定输出盘里，随时可传输于第三方跨平台阅读器。
