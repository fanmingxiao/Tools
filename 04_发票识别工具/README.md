# 发票识别工具

一款桌面应用程序，能够批量扫描目录中的发票图片，使用OCR技术识别发票关键信息，并将结果汇总导出到Excel文件。

## 功能特性

- ✅ 批量扫描发票图片（支持 jpg、png、bmp、tiff 等格式）
- ✅ 自动识别发票关键信息：
  - 发票代码
  - 发票号码
  - 金额/税额/价税合计
  - 开票日期
  - 销售方名称
- ✅ 可选递归扫描子目录
- ✅ 实时显示识别进度
- ✅ 结果预览表格
- ✅ 导出到Excel（自动汇总金额）

## 快速开始

### 1. 安装依赖

```bash
cd /Users/fanmingxiao/个人文档/Antigravity/发票识别工具
uv sync
```

### 2. 运行程序

```bash
uv run python main.py
```

> **注意**：首次运行会自动下载PaddleOCR模型（约100MB），请耐心等待。

## 使用说明

1. 点击"浏览..."按钮选择包含发票图片的文件夹
2. 根据需要勾选"包含子目录"选项
3. 点击"开始识别"按钮
4. 等待识别完成，在表格中预览结果
5. 点击"导出Excel"保存结果

## 输出格式

Excel文件包含以下列：
| 序号 | 文件名 | 发票代码 | 发票号码 | 金额(元) | 税额(元) | 价税合计(元) | 开票日期 | 销售方 | 备注 |

文件末尾自动汇总金额、税额、价税合计。

## 技术栈

- **OCR引擎**: PaddleOCR（中文识别）
- **GUI框架**: Tkinter
- **Excel处理**: openpyxl
- **图片处理**: Pillow, OpenCV

## 支持的发票类型

- 增值税普通发票
- 增值税专用发票
- 电子发票
- 其他包含标准格式的发票
